🎮 How to Use This Site

👉 Save a Website

  1. Copy any website link you like
  2. Paste it in the "Enter URL" box
  3. Give it a group name (like "Games" or "Videos")
  4. Add some #tags if you want
  5. Click the "Add URL" button - Done! 🎉

🔄 Use on Different Devices

  1. Click "Generate Sync Token" to get your special code
  2. Save this code! It's like your secret key 🔑
  3. On your other device (phone/tablet/computer):
  4. Type in the same code
  5. Click "Sync Data" - Magic! ✨

📌 Cool Things You Can Do

  • Click the 📌 to keep important links on the side
  • Type in the search box to find your links fast
  • Click the 🗑️ to remove links you don't want
  • All your links save automatically!

💡 Remember:

  • Your special sync code works on all your devices
  • Share the code with friends to share the same links
  • Everything saves automatically - no need to worry!
